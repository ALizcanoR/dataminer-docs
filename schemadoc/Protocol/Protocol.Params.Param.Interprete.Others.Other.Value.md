# Value element

Adds a numeric value to the parameter, which can be useful in case you want to show an alarm when this rare condition occurs.

## Type

integer >= 10

## Parent

[Other](xref:Protocol.Params.Param.Interprete.Others.Other)

## Examples

```xml
<Value>10</value>
```
