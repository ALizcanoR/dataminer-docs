# Decimals element

Specifies the number of decimals that will be stored in memory.

## Type

unsignedInt

## Parent

[Interprete](xref:Protocol.Params.Param.Interprete)

## Remarks

By default, 6 decimals are stored.



## Examples


```xml
<Decimals>2</Decimals>
```
