# protocol attribute

Specifies the name of the protocol that contains the parameter this parameter is linked to.

## Content Type

string

## Parent

[LinkTo](xref:Protocol.Params.Param.Mediation.LinkTo)

## Examples


```xml
<LinkTo protocol="Microsoft Platform" pid="97"/>
```
