# Factor element

Multiplies the parameter value with the specified factor.

## Type

decimal

## Parent

[Interprete](xref:Protocol.Params.Param.Interprete)

## Remarks

> [!NOTE]
> Currently, the Sequence tag is used more frequently than the Factor tag.

## Examples

```xml
<Factor>10</Factor>
```
