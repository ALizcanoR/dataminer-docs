# path attribute

When no advanced hierarchy is needed, you can use this attribute to define the table links.

## Content Type

[TypeCommaSeparatedNumbers](xref:Protocol-TypeCommaSeparatedNumbers)

## Parent

[Hierarchy](xref:Protocol.TreeControls.TreeControl.Hierarchy)

## Examples

```xml
<Hierarchy path="100,200,300"></Hierarchy>
```
