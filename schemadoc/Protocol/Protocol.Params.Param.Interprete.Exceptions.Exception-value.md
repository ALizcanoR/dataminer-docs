# value attribute

Specifies the incoming text or numeric value that you want to match with an exception value.

## Content Type

string

## Parent

[Exception](xref:Protocol.Params.Param.Interprete.Exceptions.Exception)

## Examples

```xml
<Exception id="1" value="9999">
```
