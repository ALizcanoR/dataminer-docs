# Command element

Specifies the ID of the command that will be sent when the pair is executed.

## Type

unsignedInt

## Parent

[Content](xref:Protocol.Pairs.Pair.Content)

## Remarks

In a pair, you can specify only one command.



## Examples


```xml
<Command>3</Command>
```
