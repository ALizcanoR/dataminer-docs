# dynamicIndex attribute

Specifies the display key, which can be used as a filter.

## Content Type

string

## Parent

[Group](xref:Protocol.ParameterGroups.Group)

## Remarks

The following wildcards are supported:

- \*: Matches 0 or more characters
- ? : Matches 1 character
