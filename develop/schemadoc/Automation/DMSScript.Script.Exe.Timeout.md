# Timeout element

Specifies the timeout value (in s or ms depending on the action in which it is used).

## Type

positiveInteger

## Parent

[Exe](xref:DMSScript.Script.Exe)

## Remarks

Used with script actions of type "findinteractiveclient" and "sleep".

## Examples

```xml
<Timeout>90</Timeout>
```
