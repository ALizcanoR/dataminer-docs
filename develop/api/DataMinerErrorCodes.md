# DataMiner error codes

Values are 32 bit values structured as follows:

- Sev - is the severity code
  - 00 - Success
  - 01 - Informational
  - 10 - Warning
  - 11 - Error
  - C - is the Customer code flag
  - R - is a reserved bit
- Facility - is the facility code
- Code - is the facility's status code
