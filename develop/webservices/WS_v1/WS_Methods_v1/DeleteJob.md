# DeleteJob

Use this method to delete a job.

## Input

| Item       | Format | Description                                          |
|------------|--------|------------------------------------------------------|
| connection | String | The connection ID. See [ConnectApp](xref:ConnectApp). |
| domainID   | String | The ID of the job domain.                            |
| jobID      | String | The ID of the job.                                   |

## Output

None.
