# ChangePassword

Use this method to change the password of the currently connected user.

## Input

| Item        | Format | Description                                          |
|-------------|--------|------------------------------------------------------|
| connection  | String | The connection ID. See [ConnectApp](xref:ConnectApp). |
| newPassword | String | The new password.                                    |

## Output

None.
