# AddViewToRecent

Use this method to add a view to the Recent list.

## Input

| Item       | Format  | Description                                          |
|------------|---------|------------------------------------------------------|
| Connection | String  | The connection ID. See [ConnectApp](xref:ConnectApp) . |
| ViewID     | Integer | The ID of the view to be added.                      |

## Output

None.
