# DeleteJobTemplate

Use this method to delete a job template.

## Input

| Item       | Format | Description                                          |
|------------|--------|------------------------------------------------------|
| Connection | String | The connection ID. See [ConnectApp](xref:ConnectApp) . |
| TemplateID | String | The ID of the job template.                          |

## Output

None.
