# DMAJobSuggestion

| Item    | Format | Description                                                      |
|---------|--------|------------------------------------------------------------------|
| FieldID | String | The ID of the job field.                                         |
| Value   | String | The value of the job field.                                      |
| Score   | Double | Score indicating to what extent the job field matches the input. |
